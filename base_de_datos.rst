Base de datos
=======================================================================

La mayor parte de las aplicaciones web necesitan una base de datos donde
almacenar la información que procesan. Django está preparado para trabajar con
varios tipos de bases de datos: MySql, PostgreSql, Oracle y Sqlite3. Además,
existen extensiones de terceros para trabajar con otros gestores de bases de
datos relacionales, como Firebird y Microsoft Sql Server.
